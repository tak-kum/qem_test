Detail of Inverter Parameters

PWM Carrier Period [microsecond]

The PWM carrier period indicates the frequency of the PWM output by the inverter. This frequency is limited by the inverter’s specifications. For low-voltage 24V Si-MOSFET inverters, settings up to 20kHz (50 microseconds) are possible. For high-voltage AC200V IGBT inverters, settings up to approximately 10kHz (100 microseconds) are possible.

Please verify the specifications of the inverter being used and set the value accordingly.

Dead Time [microsecond]

Specify the dead time in us (microseconds) that is described in the inverter specifications and design document. For the MCI-HV-1 inverter, 2.0 us is specified. MCI-LV-1 should be 0.5 us.

This parameter is depending on the power device (IGBT, MOS-FET) characteristics.

Detail of Design Parameter

Current PI loop Natural Frequency [Hz] and Damping Ratio

The control gain is adjusted by tuning the natural frequency and Damping Ratio for the current control system. Set the natural frequency for the current control system to be proportional to the frequency of current control. The natural frequency can be set to up to about 1/10 of the current control frequency (PWM carrier frequency). However, in many cases, a lower value may be set to leave a margin in consideration of noise during position detection and current detection. For example, if the current control frequency is 20 kHz (current control operates at 50-µs intervals), the natural frequency for the current control system can be set to 2 kHz because it can be set to a value up to 1/10 of the current control frequency. In practice, however, control at a high natural frequency may be too sensitive due to the electrical constants of the motor parameters and the frequency will often be set below 2 kHz (for example, in the range from 500 Hz to 1 kHz). For the Damping Ratio for the current control system, a value in the range from 0.7 to 1.0 is usually set. Setting a value nearer to 1.0 makes response more stable and moderate.

Speed PI loop Natural Frequency [Hz]

In the speed control module, the control gain is adjusted by tuning the natural frequency and attenuation coefficient for the speed control system. Increasing the natural frequency for the speed control system improves the responsiveness and the capability of the speed to follow the requests of the speed commands. The maximum natural frequency for speed control should be no more than 1/3 of the maximum natural frequency for current control to prevent interference with current control. If the natural frequency for the current control system is 500 Hz, the natural frequency for speed control is 500 Hz/3 = 166 Hz.

However, the program estimates the speed without using an encoder, so set a frequency lower than the natural frequency set by PLL Natural Frequency of the sensor module. The default value specifies a frequency with relatively low capability of following the requests of the speed commands to provide a margin. For example, if an interfering oscillation is at a frequency above the natural frequency, increasing the value of the natural frequency to match that of the interference may improve the capability of following the latter oscillation and driving of the motor may be more stable than with the default setting in some cases.

Speed PI loop Damping Ratio

For the Damping Ratio for the speed control system, a value in the range from 0.7 to 1.0 is usually set. Setting a value nearer to 1.0 makes response more stable and moderate. Make adjustment while checking the speed responsiveness.

Speed Observer Natural Frequency [Hz]

Set the frequency [Hz] of the PI included in the speed observer. This parameter is used when selecting the speed observer in the Speed Feedback Process.

Speed Observer Damping Ratio

Set the Damping Ratio (0.0-1.0) of the PI included in the speed observer. This parameter is used when selecting the speed observer in the Speed Feedback Process.

Speed LPF Natural Frequency [Hz]

This parameter is available when selected LPF mode. A filter is set for the estimated speed to suppress fluctuations. If the value is too small, the speed responsiveness deteriorates and the motor will not be able to follow a sudden change in speed.

Disturbance Speed Observer Natural Frequency [Hz]

Detail of Motor Sensor

Selected Slot For Control

You can select the sensor slot to use for control. As a rule, place the main sensor module in sensor slot 1. Sensor slot 2 is for auxiliary use.

Detail of General

Speed Control Period [sec]

Set to 0.0005 s (0.5 ms). If this setting needs to be changed, the AGT0 timer setting of 11.7 must also be changed to specify a new speed control interval.

Acceleration To Target Speed [rpm/s]

Set the rate at which the speed increases (acceleration rate) when a speed command value is set. The higher the value, the faster the speed will increase. When 100 is specified, the speed is increased by 100 rpm per second. In this case, the speed will reach 2000 rpm from the standstill state in 20 seconds.

Current PI Output Limit [V]

This parameter sets the limit value for the voltage command output by the current PI controller. Typically, it is set to the same value as the reference bus voltage. Adjust this parameter when you wish to control behaviour in situations such as when the current control system saturates during high-speed operation or under heavy load.

If set below the bus voltage, acceleration may not reach the speed at which normal operation is possible.

Limit of Q-axis Current [A]

This parameter limits the q-axis current. It is used when intentionally restricting torque, such as when issuing commands to the current control system. Normally, set this to either the rated current or the maximum current.

Detail of Protection

Maximum Speed [rpm]

When actually running the motor, please set the maximum speed [rpm] within the control range. If this value is set to be identical to the Limit of Over Speed, speed fluctuations may prevent control at this speed.

Limit of Over Current [A]

Set the current value with a safety margin from the maximum current value that can be output by the inverter.

Limit of Over Voltage [V]

Set the voltage at which the overvoltage protection is activated. If the inverter bus voltage exceeds the set voltage, an error occurs and the motor stops operating. Set an appropriate value according to the power supply environment used.

Limit of Over Speed [rpm]

Set the maximum speed [rpm] at which operation will halt as an error. Set the maximum permissible speed that must not be exceeded. Set this with a margin relative to the actual operating speed range.

Limit of Low Voltage [V]

Set the voltage at which the low-voltage protection is activated. If the inverter bus voltage falls below the set voltage, an error occurs and the motor stops operating. Set an appropriate value according to the power supply environment used.

Detail of Sample Delay Compensation

Detail of Voltage Error Compensation

Reference Voltage [V]

Set the reference voltage used when creating the table. As busbar voltage fluctuates during actual operation, this voltage is used to correct the table values.

Table of Voltage 1-5 [V]

Set the voltage points in the voltage error compensation table. Specifically, configure the curve around the zero-crossing point of the current using the table to correct voltage errors caused by dead time.

This table can be calculated theoretically from the PWM carrier frequency, dead time, and inverter rated current, or determined by applying a test voltage to an actual inverter.

Table of Current 1-5 [A]

Set the current point on the voltage error compensation table. This must be measured simultaneously with the above voltage point.

Detail of Motor Parameter

Pole Pairs

Set the number of pole pairs of the PM motor. The number of pole pairs is 1/2 the number of poles. Refer to the PM motor specifications.

Resistance [ohm]

For the wiring for measurement with an LCR meter, select two among the motor’s three-phase output lines U, V, and W and connect the probes to them. To measure the resistance, use the DC resistance (DCR) mode. The resistance value thus obtained is the composite resistance of the two phases, so the resistance value of the motor for one phase can be obtained by halving the composite value. Set the obtained resistance R as MOTOR_CFG_RESISTANCE in r_motor_targetmotor_cfg.h. The unit is .

Inductance of D-axis [H] and Q-axis [H]

For the wiring for measurement with an LCR meter, select two among the motor’s three-phase output lines U, V, and W and connect the probes to them. Use the series equivalent circuit mode (Ls) as the measurement mode. For detailed measurement methods, refer to the LCR meter’s instruction manual. Turn the shaft slowly and write down the maximum and minimum inductance values that are displayed. Here, 1/2 of the maximum value is Lq and 1/2 of the minimum value is Ld. Set the obtained Ld and Lq as MOTOR_CFG_D_INDUCTANCE and MOTOR_CFG_Q_INDUCTANCE in r_motor_targetmotor_cfg.h. The unit is H (henry).

Permanent Magnetic Flux [Wb]

Permanent magnets are used in the rotor of PM motors. The strength of these permanent magnets in the rotor constitutes the magnet flux. As rotational speed increases, the product of this magnet flux and the speed generates an induced voltage.

Rotor Inertia [kgm²]

Specify the moment of inertia of the motor’s rotor and shaft. The unit is kgm2. Usually, you can find the value in the documentation provided with the motor. If a load is installed, the inertia of the load should be added to the setting.

Rated Current [A]

Specify the rated current (RMS) of the motor. The unit is ampere. It is indicated on the nameplate of the motor or in the accompanying documentation.

Detail of Extended Function

MTPA

Maximum torque per current control is an algorithm used to adjust the output torque of a PM motor that has saliency (a PM motor satisfying Lq > Ld) to be the maximum possible value. This function can be enabled or disabled by this setting. In general, enable it. However, when the Ld and Lq values of the PM motor are the same or almost the same, this function cannot be used. In such cases, be sure to disable it (specify disable).

Flux-Weakening

The only parameter provided for flux weakening control is for enabling or disabling it; no other control parameters are used. The operating conditions such as the bus voltage and speed are monitored and control automatically begins when the necessary conditions are satisfied.

Speed Feedback Process

Configure processing for filtering or compensation of the speed feedback (speed detection) results. Normally, set this to Direct. If you wish to counter cogging torque, or if the speed detection value contains significant noise making control difficult, select LPF or SpeedObserver only after fully understanding the control algorithm. Unnecessary configuration may lead to unstable control.

Speed Feedback Process Description
Direct The detected speed is passed directly to the speed control system. This process should normally be selected.
LPF Apply a low-pass filter to the detected speed. This is used when ripple or similar disturbances are superimposed. Note that selecting the LPF may reduce speed response and may cause the system to malfunction during sudden load changes.
Speed Observer This is an improved version of the LPF, employing a motor model to perform estimation and interpolate speed detection, thereby reducing adverse effects on the speed control system.
