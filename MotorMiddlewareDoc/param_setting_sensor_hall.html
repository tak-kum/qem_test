Setting Parameters

From the “Sensor configuration”, you can change the settings related to H/W environment. There are three methods to calculate speed using Hall sensor signals, and you can select it from the “Speed calculation method” In the “Estimator configuration”, you can set parameters related to speed estimation.

Sensor configuration

Those parameters are intial setups that are uniquely determined due to H/W environment.

Port U/V/W phase input port

Set GPIO port for hall signal input.

Hall sensor pattern: Pattern #1 - #6

Set the pattern number of the Hall sensor. Set these based on the definition of High and Low of the detection circuit and the arrangement of the Hall sensor signal and the electrical angle.

Estimator configuration

Parameters settings for estimating speed from hall signal.

Compensation angle [rad]

An angle offset value to compensate the mounting position (mechanical offset) of the Hall sensors.

Default counts

Set a value of 80% to 100% of the “Maximum counts”.

Maximum counts

The number of counts to judge the speed as 0 (stopped). It depends on the frequency of counting, Set the number of counts equivalent to 0.25 seconds.

Speed calculation

You can choose speed calculation method from three options. The “Speed mixing deviation threshold” and “Speed mixing threshold” are parameters used in “Select Auto” in speed calculation method.

Speed calculation method

  • Select Auto : Calculate two speed estimates from the methods below, and then blends them with a dynamic weight
  • Moving Average : Calculate speed with moving average, using the latest 6 Hall edge intervals
  • Last Pulse : Calculate speed using the latest Hall edge interval

Speed mixing deviation threshold

The ripple ratio that is allowed at auto-switching of Hall sensor speed detection (range: 0.0 to 1.0). Adjust this parameter starting from 0 referring to the speed ripple ratio at the rated speed.

Speed mixing threshold [rad/s]

The speed threshold at which the speed is switched to the speed with high accuracy in the auto-switching mode. Set a value so that the actual sampling rate at this speed should separated from the bandwidth of speed control.
