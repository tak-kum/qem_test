Detail of Common Prameters

ADC_B Support

Description
Enable it for MCUs with ADC_B version of the A/D converter.
Option
Enabled, Disabled

Shared ADC support

Description
Enable when using the same ADC unit for two motor controls.
Option
Enabled, Disabled

Supported Motor Number

Description
Specify the number of motors
Set value
1 or 2

Saturation checking for PWM

Description
If selected, saturation checking code for PWM modulation is included in the build.
Option
Enabled, Disabled

Detail of General Prameters

Resolution of A/D conversion

Description
Resolution of A/D conversion

Set value * 12-bit: 0x0FFF * 14-bit: 0x3FFF * 16-bit: 0xFFFF

Detail of PWM config Prameters

GTIOCA Stop Level, GTIOCB Stop Level

Description
Set the upper PWM to the inactive level and the lower PWM to the active level. Specify according to the gate circuit of the inverter board.
Option
Pin Level Low, Pin Level High

PWM output port (UP, UN, VP, VN, WP, WN)

Description
Specifies the pin to be used for the PWM output.
Set value
See “._io.h” for the specified value.

Detail of Current and Voltage related Prameters

Current,VDC and Voltage Range

Description
Specify according to the detection circuit of the inverter board.

Counts for current offset measurement

Description
Specifies the number of times the current offset correction is calculated.
Set value
Specify a power of 2.

Offset of A/D conversion for current

Description
Specify according to the detection circuit of the inverter board. Error correction is performed in software.

Offset of A/D conversion for VDC and voltage

Description
Specify according to the detection circuit of the inverter board. Error correction changes the setting.

Detail of bootstrap config Prameters

Counts for bootstrap charge

Description
Specifies the charging time for the bootstrap circuit. Specify the number of PWM cycles.

Detail of ADC setting Prameters

Chanel and Unit of Phase current, Main Line Voltage, Phase voltage

Description
Specify the A/D channel and unit according to the CPU board used.

Adjustment delay

Description Specify the delay time by the number of states.
