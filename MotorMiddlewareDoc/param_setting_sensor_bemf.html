Setting Parameters

Parameters are broadly classified and described in two groups: those for open-loop control and those for sensorless FOC (closed-loop control using a back-EMF observer). Open-loop control is applied in the low-speed region, while sensorless FOC is applied in the medium- to high-speed region. Parameters related to switching between those control methods are included in the open-loop section.

Open-Loop

Open-loop is a method to drive motor without using feedback of speed position, applied in low speed region. The parameters to be set are described in the sections bllow. - Parameters to set current reference profile for open-loop control (Current reference profile) - Parameters relating to switching between open-loop and sensorless FOC (Switching) - Parameters relating to open-loop damping (Open-loop damping)

Current reference profile

Use the parameters below for setting the Id current reference profile in open-loop, and Iq drop when switching from sensorless FOC to open-loop control.

Open-loop Id reference [A]

Specify the current that is to flow through the d axis when open-loop control is started. This value should be no greater than the rated current. If the motor has a large moment of inertia or load, specifying a small value may lead to the motor being unable to start properly.

Id up step [step]

Specify the time required to increase the current that is to flow through the d axis when open-loop control is started. The unit of this value is the current control cycle(a value of 1 specifies the period of one current control cycle).

Id down step [step]

Specify the time required to decrease the current that is to flow through the d axis in order to switch the operation to sensorless FOC control after open-loop control has started. The unit of this value is the current control cycle (a value of 1 specifies the period of one current control cycle).

Iq down proportion

Specify the Iq reduction ratio from the reference value (used in feedback control) down to zero in open-loop mode. This Iq down proportion is applied over 256 steps of the current control cycle.

Switching

Use the parameters below for setting the switching conditions between open-loop and sensorless FOC control.

Speed threshold to FOC [rpm]

Specify the mechanical angular velocity (rpm) at which the operation is to be switched from open-loop control to sensorless FOC control. Specify a velocity that is sufficiently higher than “Speed threshold to Open-loop” parameter.

Speed threshold to Open-loop [rpm]

Specify the mechanical angular velocity (rpm) at which the operation is to be switched from sensorless FOC control to open-loop control.

Soft switch

Specify the usage of soft switching function which helps the smooth switching from open-loop to sensorless FOC control.

Transition time to FOC [s]

Specify the time for switching the operation from open-loop control to closed-loop control (sensorless FOC by using the BEMF observer). Setting this parameter too long may cause a speed drop during the transition.

Phase error to FOC [degree]

Specify the threshold value for the angle error to be used when the operation is switched from open-loop control to sensorless FOC control. The unit of this value is electrical angle (degrees). A value between 5.0 and 10.0 is generally used.

Limit of speed fb rate

Specify the default constant for calculating the limit on the compensation value for output speed commands.

Open loop damping

Use the parameters below this function which will reduce vibration of the motor under open-loop control.

Ed HPF omega [Hz]

Specify the cutoff frequency (Hz) of the HPF for estimating the voltage induced along the d axis.

Open-loop damping zeta

Specify the damping coefficient for the open-loop damping control system. Specify a value from 0.8 to 1.0.

Sensorless FOC

Sensorless FOC is a method to drive a motor in FOC, using estimated angle and speed. This module uses a BEMF observer and PLL for estimating the motor’s angle and speed, applied in middle to high speed region

BEMF Observer

BEMF observer estimates the motor’s back-EMF from the voltage commands and measured currents using an electrical motor model. The estimated BEMF is used to obtain rotor electrical angle and speed (via PLL) for sensorless FOC.

Natural frequency [Hz]

Specify the natural frequency for the BEMF observer. The frequency should be sufficiently separated from the CURRENT_CFG_OMEGA setting for the current regulator, with around 2 to 3 times of CURRENT_CFG_OMEGA as a suggested range. Specifying a value that is relatively close to CURRENT_CFG_OMEGA will make obtaining correct values for angle impossible due to oscillation of the output from the BEMF observer.

Damping ratio

Specify the attenuation coefficient for the BEMF observer. Specify 1.0 in general.

PLL

PLL is used to obtain electrical speed from the estimated BEMF.

Natural frequency [Hz]

Specify the natural frequency for the PLL for use in accumulating the angle errors obtained by the BEMF observer and in calculating angles. The frequency should be around 1/10 of that for the BEMF observer and higher than SPEED_CFG_OMEGA for the speed regulator. If an inappropriate value is specified, correct estimation of angles will not be possible and the output may oscillate.

Damping ratio

Specify the attenuation coefficient for the PLL for use in accumulating the angle errors obtained by the BEMF observer and in calculating angles. Specify 1.0 in general.
