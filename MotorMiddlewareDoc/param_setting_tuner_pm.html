How to set each parameter

This section explains how to determine each parameter in Motor Tuner PM based on motor control theory (FOC), inverter hardware, and how the parameters are actually used inside rm_motor_tuner_pm (see XML configuration).

General

PWM timer clock frequency (Hz)

Definition
Clock frequency driving the PWM timer.

Used in XML

.mcu_pwm_resolution =
    pwm_timer_clock_freq_hz * inv_info_pwm_cycle_s * 0.5

Guideline Set this to the actual timer clock after prescaler, not CPU clock.

Formula

\[ N_{\text{PWM}} = f_{\text{timer}} \cdot T_{\text{PWM}} \cdot 0.5 \]

Example

  • Timer clock = 120 MHz
  • PWM period = 50 µs

\[ N_{\text{PWM}} = 120\times10^6 \cdot 50\times10^{-6} \cdot 0.5 = 3000 \]

PWM cycle per control interrupt

Definition
Number of PWM cycles per current control interrupt.

Motor control meaning Defines the current loop sampling frequency:

\[ f_{i} = \frac{f_{\text{PWM}}}{N_{\text{tick}}} \]

Guideline

  • 1 → fastest response (recommended)
  • >1 → reduce CPU load but slows current loop

Example

  • PWM = 20 kHz
  • PWM cycle per control interrupt = 2

→ Current loop = 10 kHz

Speed control period (s)

Definition
Execution period of the speed PI loop.

Motor control rule Speed loop bandwidth should be 5–10× slower than current loop.

Formula

\[ T_{\omega} \ge 10 \cdot T_{i} \]

Example

  • Current loop = 10 kHz → 100 µs
  • Speed loop = 500 µs ✅

MCU Setting

ADC resolution

Definition
ADC counts corresponding to full scale.

Used in XML

.mcu_adc_resolution = ADC_resolution

Guideline Set exactly to MCU ADC resolution:

  • 12‑bit → 4096
  • 14‑bit → 16384

Effect Directly affects:

  • current quantization
  • observer noise floor

Inverter Setting

Current full range (A)

Definition
Maximum measurable phase current.

Used in FOC Current normalization:

\[ i_{\text{norm}} = \frac{i_{\text{meas}}}{I_{\text{full}}} \]

Guideline

\[ I_{\text{full}} \ge 1.2 \times I_{\text{rated}} \]

Example

  • Motor rated current = 12 A
  • Set full range = 16.5 A ✅

Current limit (A)

Definition
Software overcurrent limit.

Used in XML

.inv_info_current_limit

Guideline

\[ I_{\text{limit}} \le I_{\text{full}} \]

Typical:

  • 70–90% of inverter rating

Example

  • Full range = 16.5 A
  • Limit = 12 A ✅

Overvoltage threshold (V)

Definition
DC bus overvoltage protection threshold.

Guideline

\[ V_{\text{OV}} \le 0.9 \cdot V_{\text{DC,max}} \]

Example

  • 48 V system
  • Threshold = 60 V (regen margin)

Undervoltage threshold (V)

Definition
DC bus undervoltage shutdown threshold.

Guideline Set above:

  • gate driver UVLO
  • control logic minimum voltage

Example

  • 12 V system → 8 V ✅

PWM cycle time (s)

Definition
PWM switching period.

Motor control tradeoff

  • Smaller → better current ripple
  • Larger → lower switching loss

Guideline Typical PMSM:

  • 20–40 kHz → 25–50 µs

PWM dead time (s)

Definition
Dead time inserted between high/low switches.

Guideline Set from gate driver + MOSFET data:

\[ t_{\text{dead}} > t_{\text{fall}} + t_{\text{delay}} \]

Typical

  • 1–3 µs

Minimum duty cycle

Definition
Lower PWM clamp.

Purpose Avoid:

  • bootstrap UV
  • nonlinear switching region

Typical 0.05–0.1

Maximum duty cycle

Definition
Upper PWM clamp.

Purpose Reserve dead‑time margin.

Typical 0.9–0.95

Rdiff Settings (Stator Resistance Measurement)

Target current A / B

Definition
Normalized current injection levels.

\[ I = I_{\text{full}} \cdot I_{\text{target}} \]

Guideline

  • A: low current (thermal safe)
  • B: higher current (better SNR)

Example

  • Full range = 16.5 A
  • Target A = 0.6 → 9.9 A
  • Target B = 0.8 → 13.2 A

VD step wait (ms)

Definition
Wait time after voltage step.

Motor control meaning Allow current to reach steady state:

\[ t \ge 5\tau_e = 5\frac{L}{R} \]

Stabilization wait

Definition
Extra settling time before measurement.

Guideline Same unit as VD step wait.

LDQ Settings (Inductance Identification)

RLD frequency (Hz)

Definition
High‑frequency d‑axis voltage injection.

Guideline

\[ f_{RLD} \gg f_{\text{elec}} \]

but

\[ f_{RLD} < 0.2 f_{\text{PWM}} \]

Typical 100–300 Hz

LQ frequency (Hz)

Definition
q‑axis inductance injection frequency.

Guideline Higher than RLD for better separation.

Typical 800–1500 Hz

Ke Settings (Back‑EMF Constant)

Open-loop target frequency (Hz)

Definition
Final electrical frequency during open‑loop spin.

Relation

\[ \omega_e = 2\pi f_e \]

Guideline Reach region where BEMF is measurable but safe.

Open-loop ramp-up rate (Hz/s)

Definition
Acceleration of electrical frequency.

Guideline

\[ \frac{df}{dt} < \frac{T_{\max}}{J \cdot 2\pi} \]

Lower value for high inertia loads.

JD Settings (Inertia Identification)

Speed oscillation frequency (Hz)

Definition
Injected speed oscillation frequency.

Guideline Below mechanical resonance:

  • 2–10 Hz typical

Speed oscillation amplitude coef

Definition Normalized oscillation amplitude.

Tradeoff

  • Too small → poor SNR
  • Too large → nonlinear torque

Typical 0.1–0.2

Speed oscillation amplitude coef min

Definition Lower bound for adaptive reduction.

Typical 0.05

FOC Base Settings

Open-loop to Closed-loop speed threshold coefficient

Definition Transition speed from open‑loop to sensorless FOC.

Used in XML

ω_th = coef × base_speed

Guideline 0.2–0.4

Closed-loop to Open-loop speed threshold coefficient

Definition Fallback threshold for loss of observer lock.

Guideline Slightly lower than OL→CL threshold to avoid chattering.

Assumed inertia

Definition Mechanical inertia used for observer and speed PI.

Used in

  • speed observer
  • acceleration limiting

Guideline If unknown:

  • start small (1e‑6)
  • increase if oscillation occurs
